# 示例

## 示例 1：执行候选视频搜索

输入：

- `search-candidate-videos` 的任务详情。
- 产品定位和策略上下文。
- 运行时提供的搜索工具访问能力。

执行：

- 使用策略关键词和产品受众标准搜索。
- 将候选元数据保存到任务数据区域。
- 创建描述为 `候选视频列表` 的产出记录。
- 将任务标记为 `completed`。

## 示例 2：执行潜客评论任务

输入：

- `comment-on-prospects` 的任务详情。
- 上游候选视频产出记录。
- 工作上下文中的品牌语气和评论规则。

执行：

- 从上游产出加载候选数据。
- 选择符合任务说明的潜客。
- 撰写并发送上下文化评论。
- 将评论结果保存到任务数据区域。
- 创建描述为 `评论结果` 的产出记录。
- 如果必需候选数据不可用，则标记为 `failed`。

## 示例 3：执行私信跟进

输入：

- `private-message-follow-up` 的任务详情。
- 来自前序任务产出的合格潜客。
- 私信语气和行动号召指导。

执行：

- 加载合格潜客。
- 撰写简短跟进私信。
- 只通过可用受控工具发送。
- 保存已发送消息结果。
- 创建描述为 `私信跟进结果` 的产出记录。
- 按真实结果更新任务状态。
